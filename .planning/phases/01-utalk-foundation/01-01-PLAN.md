<objective>
**Create the database migration and Drizzle schema for UTalk provider integration.**

**Purpose:** Establish the data foundation for multi-provider WhatsApp support. New tables (`utalk_integrations`, `utalk_mappings`) store UTalk configuration and ID mappings. Existing tables (`conversations`, `messages`, `contacts`) gain `provider`, `external_id`, `raw_provider_data` columns so the system can distinguish Meta vs UTalk conversations and messages without breaking backward compatibility.

**Output:** A new migration file `migrations/004_utalk_integration.sql` and updates to `server/db/schema.ts` (Drizzle SQLite schema). Both must be in sync.
</objective>
